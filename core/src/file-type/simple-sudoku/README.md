# Simple Sudoku (.ss)

In a simple sudoku file, the data is stored in a human readable format.

1..|...|7..
.2.|...|5..
6..|38.|...
-----------
.78|...|...
...|6.9|...
...|...|14.
-----------
...|.25|..9
..3|...|.6.
..4|...|..2

There is an older format for Simple Sudoku files, which you may still encounter:

X6X1X4X5X
XX83X56XX
2XXXXXXX1
8XX4X7XX6
XX6XXX3XX
7XX9X1XX4
5XXXXXXX2
XX72X69XX
X4X5X8X7X

Source: SudoCue https://www.sudocue.net/fileformats.php
